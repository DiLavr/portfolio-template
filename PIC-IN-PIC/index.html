Picture in Picture
project

Picture and Picture chrome browser API is a JavaScript based interface for the Chrome browser that allows the user to view two or more websites side-by-side in a compact yet effective manner. It provides developers with an intuitive and straightforward way to integrate a picture-in-picture mode into Chrome, and can help break down walls that often stand between users and their desired content.

The JavaScript-based API helps create an efficient environment, as it requires minimal code when comparing to other solutions such as HTML Canvas or JavaScript elements. This makes it possible for developers to benefit from an optimized performance while using modern technologies - all while providing a visually appealing experience for users.

How to use?

The script allow you to capture window or chrome browser tab and stream it in Pictore - in - picture mode. It's working only in Chrome based browsers
